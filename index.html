兩百字內的個人簡介。
我叫黃庠家今年36歲來自台北，現職台北市公家機關工作6年，因工作過度僵化可替代性高，所以要轉職工程師，今年初就看彭彭老師的Youtube頻道(python的內容且看完)，但卻不知道該做出什麼作品及怎麼做，於是Google怎麼自學轉職後又去讀了碁峯的精通Python，還是覺得茫然，剛好看到彭彭老師的WeHelp計畫就來報名了，我很想要參加此計畫並成功學習後盡速完成自己的作品並成功轉職。
為了成為軟體工程師，曾做過什麼努力？
看過彭彭老師的youtube頻道、看過<<精通python>>、上網搜尋研究有哪些python的課程可以自學研讀，於是在Udemy買了一些課程如Python3: From Beginner to Pro、Master Python by Building Real-World Python Projects以及嘗試去leetcode寫一些題目(刷題)，但是仍然感覺自己在工程師的能力不足，所以也一直在找有什麼方法可以更有系統性的學習python並轉職成為前端/後端工程師。
如果參與這個計畫，會怎麼安排學習時間？
上班日每天預計花4小時於非上班時間寫程式碼+找尋程式碼內容尚未完全弄懂的部分，並利用碎片時間來上網搜尋該如何把程式碼寫得更適當或是研究新的程式碼，並每日運動半小時讓身體能夠保持在一個較好的水準來持續學習及讓思路清晰，放假時預計一天花費4小時學習新的資訊，3小時來整合過去一周學到的內容並針對尚未完全理解的部分進一步找出癥結點並攻克，花1小時鍛鍊身體來讓身體狀態保持一定的水準。

請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？
當遇到自己沒辦法馬上克服或解決的難題時，感到懊惱的情緒，我會先覺察自己的情緒來源，確認來源後思考為什麼產生這樣的情緒，若無法馬上解決問題或仍然卡住，則先看看有沒有辦法先暫時脫離當下的環境，若沒辦法離開則去喝個水上個廁所轉換一下心情，如果可以離開一下，則去稍微走動一下讓大腦轉換一下久坐而僵化的思考模式之後再度面對原本的問題可能就能轉換思路從而克服障礙。
關於這份申請網頁，請分享一個你開發時的技術心得。
一開始在自學html、CSS過程中，由於沒有接觸過CSS語法，所以不確定背景圖片的加入應該怎麼設定，而後陸陸續續看了彭彭老師的[HTML、CSS、JavaScript 網頁前端工程]裡面的影片，但仍然遇到在html中如果直接添加背景圖片則會導致背景被div的語法切割成一塊塊的，後面經由轉換思路和嘗試將背景寫在css語法下，最後得到一個連續且未被分割的背景圖片。
